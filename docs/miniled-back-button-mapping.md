# MINILED显示屏左右背键映射设置操作指南

## 操作步骤

### 1. 进入MINILED菜单
- 在MINILED显示屏上，通过手柄按键进入主菜单界面
- 菜单显示为 "GP2040-CE Mini Menu"

### 2. 导航到背键设置选项
- 使用**方向键**（UP/DOWN/LEFT/RIGHT）在菜单中导航
- 找到并选择以下选项之一：
  - **"Left Back"** - 左背键（GPIO22）
  - **"Right Back"** - 右背键（GPIO25）
- 按**B1键**（确认键）进入对应的背键映射菜单

### 3. 选择要映射的按键
在背键映射菜单中，您会看到可用的按键选项列表：
- **None** - 不映射任何按键
- **Up** - 方向键上
- **Down** - 方向键下
- **Left** - 方向键左
- **Right** - 方向键右
- **B1** - 按钮1（A键）
- **B2** - 按钮2（B键）
- **B3** - 按钮3（X键）
- **B4** - 按钮4（Y键）
- **L1** - 左肩键
- **R1** - 右肩键
- **L2** - 左扳机键
- **R2** - 右扳机键
- **S1** - 选择键1
- **S2** - 选择键2
- **A1** - 辅助键1
- **A2** - 辅助键2
- **L3** - 左摇杆按下
- **R3** - 右摇杆按下
- **FN** - 功能键
- **A3** - 辅助键3
- **A4** - 辅助键4

### 4. 确认选择
- 使用**方向键**上下滚动浏览选项
- 当前选中的选项会在显示屏上显示，选中的选项旁边会显示 **"*"** 标记
- 移动到您想要映射的按键选项
- 按**B1键**确认选择
- 选择后会自动返回主菜单

### 5. 保存设置
- 在主菜单中，导航到 **"Exit"** 选项
- 按**B1键**进入退出菜单
- 您会看到两个选项：
  - **"Save & Exit"** - 保存设置并退出（需要重启）
  - **"Exit"** - 不保存直接退出
- 选择 **"Save & Exit"** 保存您的更改
- 系统会提示确认，选择 **"Yes"** 保存并重启

## 操作提示

### 菜单导航
- **方向键上下**：在同一菜单中上下移动光标
- **方向键左右**：快速跳转（每4个选项为一组）
- **B1键**：确认/选择当前选项
- **B2键**：返回上一级菜单

### 注意事项
- 更改GPIO映射后需要**重启设备**才能生效
- 如果选择"Exit"而不保存，所有未保存的更改将丢失
- 当前选中的映射会显示 **"*"** 标记，方便识别
- 菜单支持分页显示，如果选项较多会显示"Page: X/Y"

## 示例操作流程

**设置左背键为L3（左摇杆按下）：**
1. 进入主菜单 → 选择 "Left Back" → 按B1
2. 在按钮列表中找到 "L3" → 按B1确认
3. 返回主菜单 → 选择 "Exit" → 按B1
4. 选择 "Save & Exit" → 按B1
5. 确认保存 → 设备重启后生效

**设置右背键为B1（A键）：**
1. 进入主菜单 → 选择 "Right Back" → 按B1
2. 在按钮列表中找到 "B1" → 按B1确认
3. 返回主菜单 → 选择 "Exit" → 按B1
4. 选择 "Save & Exit" → 按B1
5. 确认保存 → 设备重启后生效
